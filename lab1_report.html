可以将其写成一个函数，输入协方差阵，输出偏相关系数矩阵，如下：

> r2rp=function(R){
+   Omega=solve(R)
+   d=diag(Omega)
+   D0.5=diag(1/sqrt(d))
+   Rp=-D0.5%*%Omega%*%D0.5
+   diag(Rp)=1
+   return(Rp)
+ }

例3 相关系数在Fisher z-变换前后的收敛速度比较

> n=30
> N=10000
> rho=0.5
> all.r=NULL
> all.atanhr=NULL
> for(k in 1:N){
+   u=rnorm(n)
+   v=rnorm(n)
+   x=u
+   y=rho*u+sqrt(1-rho^2)*v
+   r=cor(x,y)
+   all.r=c(all.r,r)
+   tmp=atanh(r)
+   all.atanhr=c(all.atanhr,tmp)
+ }
> par(mfrow=c(2,2))
